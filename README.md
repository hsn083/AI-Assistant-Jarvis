# 🤖 Jarvis AI Assistant

An advanced AI-powered personal assistant that helps users interact with artificial intelligence through voice commands, smart automation, and real-time AI responses.

Jarvis AI Assistant is built to provide an intelligent assistant experience by combining AI models, APIs, automation tools, and modern software architecture.

# ⚙️ Installation Guide

## Step 1: Clone Repository

```bash
git clone https://github.com/hsn083/AI-Assistant-Jarvis.git
```

Go inside project folder:

```bash
cd AI-Assistant-Jarvis
```

---

# 🐍 Backend Setup

Open terminal:

```bash
cd Backend
```

Create virtual environment:

```bash
python -m venv venv
```

Activate environment:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# 🌐 Frontend Setup

Go to frontend folder:

```bash
cd Frontend
```

Install packages:

```bash
npm install
```

---

# 🔑 Environment Variables Setup

Create a file:

```
.env
```

Add your API keys:

```env
AI_API_KEY=your_api_key_here

HUGGINGFACE_TOKEN=your_token_here
```

⚠️ Never upload your `.env` file to GitHub.

Keep your API keys private.

---

# ▶️ Run Project

## Start Backend

```bash
python main.py
```

---

## Start Frontend

```bash
npm run dev
```

# 🐛 Common Problems

## Dependency Error

Run:

```bash
pip install --upgrade pip
```

Then:

```bash
pip install -r requirements.txt
```

---

## API Not Working

Check:

- `.env` file exists
- API keys are correct
- Internet connection is available
